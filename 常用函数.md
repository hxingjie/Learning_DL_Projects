## 绘图
```python
import numpy as np
import matplotlib.pyplot as plt

x = np.arange(0, 6, 0.1, dtype=float)  # [0, 6), 步长为0.1
y1 = np.sin(x)
y2 = np.cos(x)

plt.plot(x, y1, linestyle="-", label="sin")
plt.plot(x, y2, linestyle="--", label="cos")
```
